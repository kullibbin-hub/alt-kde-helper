# Alt KDE Helper

## Описание программы

Программа предназначена для настройки и обслуживания операционных систем Alt Linux 11 с рабочим столом KDE6 — KWorkstation и особенно Starterkit, который можно одним кликом привести в полностью готовый для работы дистрибутив, оставаясь быстрым и лёгким, но полностью функциональным.

**Программа позволяет:**

- включать отключённые опции (например, загрузку виджетов и тем)
- получить sudo и права на его использование
- загрузить кодек openh264 без применения VPN
- выбрать зеркала репозитория (есть выбор самого быстрого по результатам теста)
- включить создание миниатюр для значков файлов 3D-моделей, FreeCAD и AutoCAD
- глубокую очистку кэшей — в домашнем каталоге, для Flatpak-приложений, аккуратную очистку системного кэша, удаление старых журналов и сжатие до размера 100 МБ максимум, очистку кэша пакетов (без применения небезопасной опции autoremove), удаление неиспользуемых рантаймов Flatpak, удаление старых ядер
- и многое другое

Кнопка **«Выбрать рекомендованные настройки»** одним кликом включает набор настроек, которые обычно делает каждый пользователь после установки системы.

## Установка

### Простой способ (рекомендуемый)

1. Нажмите зелёную кнопку **Code** вверху этой страницы
2. Выберите **Download ZIP**
3. Распакуйте архив в папку `Загрузки`
4. Откройте распакованную папку `alt-kde-helper`
5. Правый клик по файлу `install.sh` → **Свойства** → **Права** → поставьте галочку **«Разрешить выполнение файла как программы»**
6. Правый клик по файлу `install.sh` → **Запустить исполняемый файл**
7. Откроется окно терминала, где будет виден процесс установки
8. После завершения нажмите любую клавишу — терминал закроется

**После установки программа появляется в меню приложений KDE в категории «Система»**

### Удаление

1. Правый клик по файлу `uninstall.sh` → **Свойства** → **Права** → поставьте галочку **«Разрешить выполнение файла как программы»**
2. Правый клик по файлу `uninstall.sh` → **Запустить исполняемый файл**

Будут удалены не только файлы программы, но и сохранённая конфигурация в профиле пользователя.

## Как пользоваться

1. Выберите нужные действия на вкладках **«Настройки»** и **«Обслуживание»**
2. Нажмите кнопку **«Применить»**
3. Откроется терминал, в котором будут выполняться указанные действия
4. После выполнения всех действий появится сообщение **«Нажмите любую клавишу»**
5. Нажмите любую клавишу или закройте окно терминала
6. Кнопка **«Выбрать рекомендованные настройки»** автоматически включает чекбоксы рекомендуемых автором настроек. При повторном нажатии — чекбоксы снимаются
7. Многие настройки правильно применяются после перезахода в сеанс, а обновления — после перезагрузки компьютера

## Особенности

- Действия с откатом имеют два чекбокса: **«Применить»** и **«Откат»**
- Откат доступен только после того, как действие было применено
- Выбор зеркала репозитория — радио-кнопки, автоматически удаляет другие варианты из очереди
- **Важно:** Если выбранное зеркало (например, самое быстрое или Yandex) не проходит проверку, автоматически возвращается зеркало по умолчанию — Р11
- sudo устанавливается и настраивается для пользователя сразу в процессе установки программы
- Уже применённое действие выделяется зелёным цветом шрифта, но его можно ещё раз применить при необходимости

## Установка драйверов NVIDIA

Инструкция: [https://alt-kde.wiki/graphics/nvidia/nvidia-drivers/](https://alt-kde.wiki/graphics/nvidia/nvidia-drivers/)

Общая вики Alt KDE: [https://alt-kde.wiki/wiki/](https://alt-kde.wiki/wiki/)

Процесс установки драйверов NVIDIA требует ответов на вопросы в терминале и перезагрузки, поэтому в данной программе не реализован.

**Рекомендованный способ из инструкции** — установка через `epm`.

В данной программе `epm` можно установить двумя способами:
- Пункт **«Установить / обновить eepm»** (на вкладке «Обслуживание»)
- Пункт **«Установка рекомендованных пакетов»** (на вкладке «Настройки»)

В обоих случаях `epm` установится автоматически.
